Personal Portfolio Website
Frontend Development - Beginner Project

This project is a simple, clean, and responsive personal portfolio website created using HTML and CSS. 
It serves as a centralized hub to showcase my background, skills, and contact information.

Features:
Responsive Design: Optimized for viewing on both desktop and mobile devices.
Clear Navigation: Simple header with "About" and "Contact" links for easy navigation.
Professional Introduction: A dedicated "About" section featuring a profile image and a concise professional summary.
Direct Contact: A dedicated "Contact" section with multiple ways to Get In Touch (Email, Phone, LinkedIn, and GitHub).

Technologies Used:
HTML5
CSS3
